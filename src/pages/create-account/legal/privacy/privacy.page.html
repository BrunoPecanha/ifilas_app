Política de Privacidade

Política de Privacidade – iFilas

Última atualização: 16 de fevereiro de 2026

1. Controlador dos Dados

O controlador dos dados pessoais tratados pelo aplicativo iFilas é BMP Labs - Building Modern Products.

Para exercer seus direitos relacionados à proteção de dados, entre em contato pelo e-mail: brunoitdesenv@gmail.com.

2. Dados Coletados

Coletamos os seguintes dados pessoais:

  • Nome e sobrenome;
  • E-mail;
  • Telefone;
  • Endereço, CEP, cidade e estado;
  • Dados de uso da plataforma.

3. Finalidade do Tratamento

Os dados são utilizados para:

  • Criar e gerenciar contas;
  • Permitir agendamentos;
  • Enviar notificações relacionadas ao serviço;
  • Melhorar a experiência do usuário;
  • Cumprir obrigações legais e regulatórias.

A base legal para o tratamento é a execução de contrato e o consentimento do titular, conforme a Lei Geral de Proteção de Dados (LGPD).

4. Compartilhamento de Dados

Os dados poderão ser compartilhados com:

  • Estabelecimentos onde o usuário realiza agendamentos;
  • Provedores de infraestrutura tecnológica;
  • Autoridades legais, quando exigido por lei.

5. Armazenamento e Segurança

Adotamos medidas técnicas e administrativas para proteger os dados pessoais contra acessos não autorizados, perda, alteração ou divulgação indevida.

6. Direitos do Titular

Nos termos da LGPD, o titular pode solicitar:

  • Confirmação da existência de tratamento;
  • Acesso aos dados;
  • Correção de dados incompletos ou desatualizados;
  • Anonimização, bloqueio ou eliminação;
  • Portabilidade dos dados;
  • Revogação do consentimento.

7. Retenção dos Dados

Os dados serão mantidos enquanto a conta estiver ativa ou pelo período necessário para cumprimento de obrigações legais.

8. Alterações

Esta Política poderá ser atualizada periodicamente. Recomendamos que o usuário revise este documento regularmente.
